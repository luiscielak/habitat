Action Variants:
================

[Show my meals so far]
- Context: Full meal log display
- Input: None (display only)
- Output: Detailed meal list with macros

[Workout recap]
- Context: Today's activity summary
- Input: Intensity, type, duration, notes
- Output: Recovery meal recommendations

[I'm hungry]
- Context: Current intake summary
- Input: Filters, notes
- Output: Meal recommendations

[Plan a meal]
- Context: Remaining macros
- Input: Eating out / For week / For today
- Output: Meal plan with options

[Quick sanity check]
- Context: Full day summary
- Input: Check type (Nutrition/Fitness/General)
- Output: Status assessment + suggestions

[Unwind]
- Context: Day completion status
- Input: Anything since dinner?
- Output: Day closure + next day prep

Annotations:
-----------
A1: Recommendations are numbered and specific
    Each includes calorie and protein estimates
    Ordered by relevance to current intake

A2: Context card is collapsible to save space
    Shows expanded by default, can minimize

A3: Quick filters are multi-select chips
    Gradient border when selected

A4: Gradient button is full-width, prominent
    Loading state shows spinner + "Thinking..."

A5: "×" dismisses entire coaching flow
    Returns to Dashboard

Gestures:
---------
- Swipe down: Dismiss coaching
- Tap "×": Dismiss coaching
- Tap filter: Toggle selection
- Tap expand: Show/hide context details

Flow Hooks:
-----------
- Submit → Loading → Response
- "Try Another" → New request with same context
- "Done" → Dashboard
- Dismiss → Dashboard (maintains last state)
