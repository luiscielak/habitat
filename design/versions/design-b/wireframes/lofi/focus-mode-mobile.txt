Annotations:
-----------
A1: Large checkbox (64×64pt visual, 88pt touch target)
    Gradient border when unchecked
    Gradient fill + checkmark when checked
    Scale animation on toggle (1.0 → 1.2 → 1.0)

A2: Dot indicators show position in habit sequence
    Current habit highlighted with gradient dot
    Swipe gestures navigate between habits

A3: Time input appears only for meal habits
    Tapping opens native iOS time picker

A4: "×" button exits to Dashboard
    Swipe down also exits (gesture alternative)

Gestures:
---------
- Tap checkbox: Toggle completion state
- Swipe left: Next habit
- Swipe right: Previous habit
- Swipe down: Exit to Dashboard
- Tap "×": Exit to Dashboard
- Tap time: Open time picker

Flow Hooks:
-----------
- Complete last habit → Perfect Day celebration
- Tap "Back to Home" → Dashboard
- Swipe past last habit → Dashboard
- Exit → Dashboard (maintains state)
