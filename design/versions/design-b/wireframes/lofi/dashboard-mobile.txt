Annotations:
-----------
A1: Progress ring is interactive - tap to launch Focus Mode
    Ring stroke uses gradient (purple → pink)
    Center shows completion count with large typography

A2: "Continue" button is gradient-filled, launches Focus Mode
    starting at first incomplete habit

A3: Habit group cards show category summary
    Dot indicators show completion status per habit
    "Next" shows the first incomplete habit in group

A4: Coaching CTA uses glass/blur background
    Gradient button for primary action

States:
-------
- Empty (0/9): Ring empty, all dots gray, "Start your day" CTA
- Partial (6/9): Ring partial fill, mixed dots
- Complete (9/9): Ring full + glow, "Perfect Day!" celebration ✨
- Loading: Skeleton loaders for ring and cards

Flow Hooks:
-----------
- Tap Progress Ring → Focus Mode (first habit)
- Tap "Continue" → Focus Mode (first incomplete)
- Tap Group Card → Expanded group detail
- Tap "View All" → Daily View (list mode)
- Tap "Ask Coach" → Coaching Experience
- Tab: Daily → Focus Mode
- Tab: Week → Weekly View
