Annotations:
A1: Date header shows current date, previous/next arrows for navigation
A2: "(Today)" indicator shown when viewing current date
A3: Checkboxes toggle on tap with haptic feedback
A4: Time inputs (10:30 AM) are tappable, open time picker sheet
A5: Pre-workout meal only appears on workout days (conditional)
A6: Completion counter updates in real-time (6/9 completed)
A7: "Today" button jumps to current date
A8: Empty state: All checkboxes unchecked, counter shows 0/9
A9: Full completion: All checkboxes checked, counter shows 9/9
A10: Past day view: Same layout, shows historical completion state

Flow Hooks:
→ Weekly View (via tab bar)
→ Home View (via tab bar)
→ Time Picker Modal (tap time input)
↑ Tap checkbox → Toggle completion, update counter
↑ Tap time → Open time picker sheet
↑ Tap "Today" → Jump to current date
